Jogo de plataforma com foco em movimentação, sem combate, podendo pular pelas paredes, plataforma que tem movimentação e objetos que contam como espinhos que causa dano no player.
Como não possui sprite, para poder ver as colisões: click na opição de depuração e ative a opição "Formas de Colisão Visíveis".

Este trabalho tem como foco enternder a linguagem de programação GDScript.
